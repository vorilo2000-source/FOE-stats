# FOE Stats

Forge of Empires Guild Statistics Dashboard

## Doel

FOE Stats is een webapplicatie voor het analyseren van Forge of Empires-gildedata. Het dashboard biedt inzicht in spelersprestaties, gildestatistieken, Guild Battlegrounds (GBG), Guild Expedition (GE), voortgang en historische trends.

## Functionaliteiten

### Dashboard

* Gilde-overzicht
* Actieve spelers
* Totaalscores
* Dagelijkse wijzigingen
* Historische trends

### Spelers

* Ranglijsten
* Activiteitsoverzicht
* Groei-analyse
* Historische scoreontwikkeling

### Guild Expedition

* GE voortgang
* Deelnamepercentages
* Spelerbijdragen
* Historische vergelijking

### Guild Battlegrounds

* GBG statistieken
* Sectorbijdragen
* Gevechten
* Rankings

### Data Import

* Handmatige import
* JSON bestanden
* CSV bestanden
* Lokale opslag

### Rapportages

* Export naar CSV
* Export naar Excel
* Afdrukbare rapporten
